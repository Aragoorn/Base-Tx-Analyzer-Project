# Base-Tx-Analyzer

A professional TypeScript utility designed to interact with the Base network and analyze on-chain data programmatically.

## Features
- **Network Connectivity:** Uses `ethers.js` to establish a secure connection with the Base mainnet.
- **On-Chain Analysis:** Fetches and logs real-time blockchain data (e.g., latest block information).
- **Type Safety:** Built with TypeScript to ensure reliable and clean code execution.

## Tech Stack
- TypeScript
- Ethers.js
- Node.js
